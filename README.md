# 🐾 Colmillo Natural – Tienda Web

## Archivos incluidos
- `index.html` — La tienda completa (todos los productos, carrito, checkout)
- `server.js` — Servidor Express para Render
- `package.json` — Dependencias

---

## 🚀 Cómo subir a Render

### 1. Sube a GitHub
1. Crea un repositorio nuevo en [github.com](https://github.com) (puede ser privado)
2. Sube estos 3 archivos: `index.html`, `server.js`, `package.json`

### 2. Deploy en Render
1. Ve a [render.com](https://render.com) y crea una cuenta gratis
2. Click en **"New +"** → **"Web Service"**
3. Conecta tu repositorio de GitHub
4. Configura así:
   - **Name:** colmillo-natural
   - **Runtime:** Node
   - **Build Command:** `npm install`
   - **Start Command:** `node server.js`
   - **Instance Type:** Free
5. Click **"Create Web Service"**

Render te dará una URL tipo: `https://colmillo-natural.onrender.com`
